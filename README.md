# Smiski Tracker

A collection tracker for [Smiski](https://smiski.com/) figures. Browse every series, mark which figures you own, and see totals for what you've collected versus what's missing. Sign in to sync your collection to the cloud; otherwise it's saved locally in the browser. Powered by [Supabase](https://supabase.com/).

## Deployment

Pushes to `main` are built and deployed to GitHub Pages via [`.github/workflows/deploy.yaml`](.github/workflows/deploy.yaml).
